# Sprint 1 color exploration — record

Process record for the M4 color decision (2026-07-06). Files here are the
verbatim designer outputs and persona-panel scores; the visual boards live in
the M4 artifact (all rounds preserved in its version history) and the
claude.ai/design project "Oravan Color Exploration (M4)".

## Process

1. **Round 1 (anchored, superseded):** three kickoff-defined candidates on the
   Oravan-kit forest ground + advisory panel. Rejected by founder: the point
   was to sandbox out previous builds, and renders lacked the real logo work.
2. **Round 2 (mixed, superseded):** blind-pass candidates added alongside the
   anchored ones; anonymized panel. Same objection — the anchored frame still
   dominated the board.
3. **Round 3 (standalone, decisive):** seven palette-blind designers across
   seven aesthetic seeds (one near-duplicate dropped); six candidates rendered
   live with the real Oravan mark/wordmark; computed WCAG AA audit on every
   token pair; fresh 7-persona panel scored anonymized shuffled labels.

## Round 3 result (panel means / first picks)

- **G · Field Notebook 81.9 / 1 — CHOSEN (M4, founder)**
- H · Kitchen-Table Wire Service 83.1 / 2
- E · Broadsheet Register 81.0 / 3 (flagged: terracotta red-adjacency, 3 of 7 seats)
- J · Registrar's Ledger 78.3 / 0
- F · Reading Room 77.7 / 0
- K · Herbarium Glasshouse 73.4 / 1 (the standalone-derived green; eco-coding flags)

Decision rationale: G was the only top candidate with zero nonpartisan or
legibility flags — second-place nearly everywhere, the budget-holder seat's
first pick, and the safest fit for the nonpartisan-by-construction hard rule.
