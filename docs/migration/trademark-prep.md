**Why this probably doesn't block a Class 42/41/45 filing, but still matters:** dental devices and civic-engagement software are different goods, channels, and buyers — the core likelihood-of-confusion factors point away from conflict. But the marks are *identical*, not merely similar, which is the factor that cuts hardest; identical-word coexistence across classes doesn't insulate against real-world brand confusion, SEO collisions, or a contested opposition. The attorney should pull the actual USPTO record and make the call.

## Filing recommendation *(options for the attorney — not decided here)*

- **Mark:** file the **ORAVAN standard-character wordmark** first (protects the word in any font/color — usually the highest-value first filing). The stylized logo/wordmark can follow as a separate **design mark** (scope-vs-cost call).
- **Nice class candidates** (attorney sets final class + exact goods/services wording):
  - **Class 42** (likely primary) — "providing a website featuring non-downloadable software for identifying users' elected representatives and summarizing pending legislation." Where comparable civic-tech web tools sit.
  - **Class 41** — "providing information in the field of government and civic affairs; plain-language explanations of pending legislation."
  - **Class 45** — some civic platforms classify the "contact your rep" action here; draft carefully given nonpartisan-by-construction (avoid anything reading as political "advocacy").
  - **Class 9** — not needed for a web app today; relevant if a native mobile app ships.
  - A **multi-class filing** (42 + 41) is worth discussing.
- **ITU basics:** intent-to-use (Lanham §1(b)) lets you file on bona-fide intent before actual use — fits filing ahead of/with launch. Process: file → serial # + examining attorney → **examination** (~5–5.6 mo to first action per USPTO's dashboard; ~11.7 mo avg total pendency) → **publication** (30-day opposition window) → **Notice of Allowance** → **Statement of Use** (or extensions, up to 24 mo) → registration. Fees (2025 — confirm live): **$350 base/class**, possible surcharges ($100–$200/class), **$150/class** later for the Statement of Use. Filing should follow the domain being settled — it is (oravan.org is live).
